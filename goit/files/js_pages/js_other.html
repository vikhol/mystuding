Вместо выражения "bubbles = bubbles + 1" можно (и нужно) использовать "bubbles++"
А вместо выражения "bubbles = bubbles - 1" нужно использовать "bubbles--"

Cсылки на свойства и методы объектов можно объединять в цепочки (через точки), чтобы избежать создания лишних временных переменных. Пример: var index = ship.locations.indexOf(guess);

Методы и св-ва для строк

Строку можно рассматривать как массив, элементами которого являются отдельные символы, а индексы начинаются с 0 (как и в обычных массивах).
  • length - содержит кол-во символов в строке (пробел считается за символ).
  • indexOf - метод получает строку-аргумент и возвращает индекс первого символа первого появления аргумента в своей строке. А если значение не найдено, он вернет -1 (Пример 1 в other.js). Массивы тоже поддерживают метод indexOf! Он получает значение и возвращает индекс этого значения, если тот присутствует в массиве, или -1, если отсутствует.
  • lastIndexOf Аналогичен indexOf, но находит последнее вхождение (вместо первого).
  • charAt() возвращает указанный символ из строки.
  • substring получает 2 индекса, после чего возвращает заключенную между ними строку. (Пример 2 в other.js)
  • split получает символ-ограничитель и разбивает строку на части по позиции ограничителя. (Пример 3 в other.js)
  • toLowerCase Возвращает строку, в которой все символы верхнего регистра преобразуются к нижнему регистру.
  • toUpperCase Возвращает строку, в которой все символы нижнего регистра преобразуются к верхнему регистру.
  • concat Соединяет строки.
  • replace Находит подстроки и заменяет их другой строкой.
  • slice Возвращает новую строку, из которой удалена часть исходной строки.
  • match Ищет совпадения регулярного выражения в строке.
  • trim Удаляет пропуски с обоих концов строки. Метод удобен при обработке данных, вводимых пользователем.
