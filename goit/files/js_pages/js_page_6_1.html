Глава 6

DOM (Document Object Model)

Страничка для практики.

Ключевые моменты

  • Браузер создает модель DOM страницы в процессе загрузки и разбора разметки HTML. Поэтому нужно либо помещать скрипт в конец body, либо воспользоваться св-вом window.onload и присвоить ему нужную ф-ю. (Пример в dom.js)
  • Для получения доступа к DOM из нашего кода. Мы используем встроенный объект document, он содержит целый набор свойств и методов для модификации DOM
  • Метод document.getElementById получает элемент из DOM по идентификатору и возвращает объект element, представляющий элемент страницы.
  • getElementsByClassName используется для получения элементов по имени класса.
  • getElementsByTagName возвращает все элементы с заданным именем тега в виде объекта типа NodeList с которым можно работать как с массивом.
  • Чтобы полученный эл-т можно было использовать в коде, его обычно присваивают какой-нибудь переменной. (var firstPar = document.getElementById("paragraf_1");)
  • Свойство innerHTML содержит текстовое содержимое элемента, а также всю его вложенную разметку HTML и может использоваться для изменения содержимого элемента. firstPar.innerHTML = "Параграф изменен";
  • innerHTML представляет все содержимое элемента, включая вложенные элементы (например, абзац помимо текста абзаца может включать элементы em и img)
  • Запрашивая элемент из DOM методом getElementById, мы получаем объект element, который может использоваться для чтения, изменения, замены содержимого или атрибутов элемента.

    • Как задать атрибут элемента методом setAttribute

    (Пример в dom.js)

  • Метод getAttribute позволяет получать значения атрибутов элементов.
  • Метод setAttribute позволяет задавать значения атрибутов элементов.
  • paragraf_2.setAttribute("class", "greentext"); - задаст для эл-та "paragraf_2" атрибут "class" и присвоит ему значение "greentext".
  • Вызов setAttribute для любого объекта element изменяет значение существующего атрибута, а если атрибут не существует — добавляет новый атрибут в элемент.

    • Значения атрибутов можно читать.

  • Для чтения атрибутов эл-та используется метод getAttribute (Пример с описанием в dom.js)
