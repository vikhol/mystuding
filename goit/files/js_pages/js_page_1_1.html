Глава 1

Операторы

  • = используется для присваивания значения переменной
  • == проверяет равны ли два значения (используется при проверке)
  • != проверяет, что два значения не равны
  • >= больше или равно
  • <= меньше или равно
  • > больше
  • < меньше

    • Сокращенные арифметические операторы

  • count++ увеличивает значение count на 1
  • count-- уменьшает значение count на 1
  • count += x равносильно выражению count = count + x
  • count -= x равносильно выражению count = count - x
  • count *= x равносильно выражению count = count * x
  • count /= x равносильно выражению count = count / x

    • Логические операторы

    || (ИЛИ) - "если хотя бы один из аргументов true, то возвращает true, иначе – false"
    && (И)- возвращает true, если оба аргумента истинны, а иначе – false:
    ! (НЕ) - проверяет на НЕравенство (!=) или на false. Например: if (!peremennaya) {...}

    Циклы

    If...Else

    if получает условие, и если оно верно выполняет блок кода.

    else - "неверное условие" необязательный блок, добавляется после блока if и выполняется только в том случае, если условие if оказалось ложным.

    while

    while - цикл-петля. Он будет выполняться снова и снова пока истинно его условие while (условие) { блок кода }

    do...while

    do..while - цикл, описанный, таким образом, сначала выполняет тело, а затем проверяет условие. do { // тело цикла } while ( условие );. Если условие while верно, тело цикла будет выполнено еще раз.
