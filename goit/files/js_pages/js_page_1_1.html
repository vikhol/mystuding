Операторы

= - используется для присваивания значения переменной

== - проверяет равны ли два значения (используется при проверке)

!= - проверяет, что два значения не равны

>= - больше или равно

> - больше

Логические операторы

|| (ИЛИ) - "если хотя бы один из аргументов true, то возвращает true, иначе – false"
&& (И)- возвращает true, если оба аргумента истинны, а иначе – false:
! (НЕ) - а вот с этой функцией надо бы получше разобраться.

Цикл while

var scoops = 3;
while (scoops > 0) {
document.write("Эх раз, да еще раз!");
scoops = scoops - 1; }
document.write("Достаточно");

alert("Два");
