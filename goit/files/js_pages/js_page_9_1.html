Обработка событий

Пример с картинками.

Упражнение с картой сокровищ

Таймер

Гоша Дударь

Обработчик - фрагмент кода, который должен выполняться при возникновении какого-либо события. Т.е. это обычная функция (часто такие ф-и называют "функцией обратного вызова")

  • Чтобы обработчик сработал существуют разные способы регистрации и вызова этого о-чка.
  • Чтобы назначить смену аттрибута src, нужно использовать метод element.src = "../новый/путь/к файлу".

    • Объект события

  • Чтобы обработчик работал по нужному элементу, ему передается объект события.
  • Объект события содержит общую инфу о событии: какой эл-т породил это событие, в какое время оно произошло и т.д.. Например, для щелчка мыши передаются координаты щелчка
    • В примере с картинками (imggame2.js) Происходит следующее:
    1. Мы щелкаем на изображении
    2. происходит событие щелчка
    3. для этого события создается объект события
    4. который уже передается функции обработчика

    В обработчике можно использовать этот объект и его св-ва.

  • .target - св-во в котором хранится ссылка на объект, сгенерировавший событие.
    • Например, если пользователь щелкнул по эл-ту страницы, этот эл-т становится источником события и к нему можно обратиться следующим образом function showAnswer(eventObj) {var image = eventObj.target;}
  • .id содержит id объекта события
  • clientX, clientY: смещение (в пикселах) указателя мыши от левого (и верхнего) края окна браузера.
  • screenX, screenY: смещение (в пикселах) указателя мыши от левого (и верхнего) края пользовательского экрана.
  • pageX, pageY: смещение (в пикселах) указателя мыши от левого (и верхнего) края страницы браузера.

    • Таймеры

    setTimeout - "секундомер", который по истечении таймера вызовет отправленную ему ф-ю.
  • время задается в милисекундах (в 1-й секунде 1000 милисекунд)
  • setTimeout(myFunctionName(), 5000); через пять секунд запустит ф-ю myFunctionName (если та вообще существует).
    • setInterval - таймер-цикл. Он задает интервал через который ф-я будет вызываться снова и снова.
  • setInterval(myFunctionName(), 5000); будет запускать ф-ю myFunctionName каждые 5 секунд.
  • Чтобы остановить счетчик setInterval используется встроенная ф-я clearInterval(), ей передается имя переменной, которой был назначен таймер setInterval. Пример:
    • var id = setInterval(myFunctionName, 1000); запускает таймер
    clearInterval(id); этот таймер останавливает.

    Очереди и события

  • Все события в браузере отслеживаются и обрабатываются.
  • Обработка событий происходит по очереди
  • Браузер извлекает события из очереди и передает их подходящему обработчику, если он имеется.
  • По возможности обработчики должны быть короткими и эффективными. А иначе очередь переполнится и браузер начнет тормозить.
