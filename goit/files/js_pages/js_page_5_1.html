Глава 5

Объекты

Пример "работы" объекта

  • Определение объекта начинается с открывающейся фигурной скобки
  • Объявление свойства состоит из имени, двоеточия, и значения
  • Свойства разделяются запятыми
  • Закрывается все это фигурной скобкой и точкой с запятой };
  • Если строка, используемая как имя свойства, содержит пробелы, ее необходимо заключить в кавычки: "on sale": true
  • Объект не может содержать два свойства с одинаковыми именами.
  • После значения последнего свойства запятая не ставится.

    • Как использовать св-ва объекта

  • Чтобы обратиться к св-ву объекта, нужно сперва указать имя объекта, а затем после точки нужное св-во. Пример: taz2107.power = 200; - изменит св-во power и задаст ему значение 200
  • Есть и более гибкий способ обращения к св-ву: taz2107["power"] = 300 - так же изменит св-во power и присвоит ему значение 300 (Пример #2 в objects.js)
  • Если ранее этого свойства в объекте не существовало, то оно добавится в объект. taz2107.needsWashing = true; - добавит в объект taz2107 св-во needsWashing и задаст для него значение true
  • Для удаления свойств используется команда delete. Таким образом delete taz2107.needsWashing; - удалит св-во needsWashing из объекта taz2107 (При удалении уничтожается именно само св-во, а не только его значение и обратиться к нему потом уже не получится.)
  • Чтобы вывести все св-а объекта (в консоль, например), нужно просто использовать имя объекта. console.log(taz2106); Но лучше воспользоваться циклом for in (Пример #1 в файле objects.js) Такой способ выводит в консоль все тело объекта (с комментами)
  • В остальном, свойства объекта используется так же, как переменные.

  • Чтобы создать объект без св-в (для заполнения в будущем) нужно просто пропустить все св-ва. Пример: var taz2109 = { };

  • Все изменения свойств отражаются на объекте, который был передан функции при вызове. (пример в файле objects.js, функция "skrutkaProbega")

    • "Методы"

  • Методами называются функции встроенные в тело объекта.
  • Для запуска функции из объекта (т.е. метода) так же используется запись через точку. taz2107.drive();
  • Добавить метод в объект можно командой object.svoistvo = function() {...}; (по аналогии с добавлением свойств)
  • Ф-ия "метод"" не имеет названия
  • Чтобы использовать в "методе" какое-либо св-во данного объекта, нужно использовать команду this. Пример: stop: function() {this.started = false;} или if (this.started) {
  • Значение this присваивается при вызове метода.

    • Методы могут все то же, что и обычные функции:

    1. Содержать в себе локальные переменные
    2. Возвращать значения
    3. Работать с аргументами

    Встроенные объекты

    В JavaScript присутствуют и готовые дефолтные объекты

    Например:

    • Date - предназначен для работы с датой и временем.
    • Math - используется для работы с числами (генерация и т.д.)
    • RegExp - позволяет искать текст в строках по заданному шаблону.
    • JSON - с помощью этого о-та можно передавать о-ты в другие приложения.
    • console - метод log объекта console использовался для вывода сообщений на консоль.
    • document - используется для записи в веб-страницу из программного кода
    • Window - представляет свойства, относящиеся к браузеру, и методы, которые могут использоваться в вашем коде.
