Анонимные ф-и, область действия и замыкания.

Анонимные ф-и

  • При определении ф-и в виде функционального выражения присваивать имя ф-и не обязательно.

    • Вместо:

    function handler() { alert("Yeah, that page loaded!"); }

    window.onload = handler;

    Можно использовать:

    window.onload = function() { alert("Yeah, that page loaded!"); };

  • Анон. ф-и можно использовать для сокращения записи некоторых событий.
  • Если какому-то коду или событию нужно прописать ссылку на какую-то ф-ю, то вместо этого можно записать сразу функционально выражение.

    • Например, вместо

    function cookieAlarm() {

    alert("Time to take the cookies out of the oven");

    };

    setTimeout(cookieAlarm, 600000);

    Можно использовать сразу

    setTimeout(function() {

    alert("Time to take the cookies out of the oven");

    }, 600000);

    Как вложения влияют на область действия ф-й.

    Пример в oblastdest.js

  • Ф-и, определяемые внутри др. ф-й, обладают локальной видимостью.
  • Вложенные ф-и, созданные путем обычного объявления, видны в любой точке родительской ф-и.
  • Если при создании вложенной функции используется функциональное выражение, то эта вложенная функция определена только после обработки функционального выражения.

    • Лексическая область действия

    Пример 2 в oblastdest.js

  • Если ф-я содержит локальную переменную, которая совпадает с глобальной по имени, но не совпадает по значению, ф-я будет использовать значение локальной переменной.
  • Все локальные переменные хранятся в "окружении" (environment).

    • Ф-ии JavaScript всегда выполняются в том же окружении, в котором они определяются. Чтобы узнать откуда внутри функции появилась переменная, нужно провести поиск во внешних функциях, вверх по вложениям.

    Замыкания.

    Замыкание состоит из функции и используемого ею окружения.
  • В коде ф-и могут присутствовать локальные переменные (включая параметры ф-и), а так же переменные, не определенные локально (а где-то за пределами ф-и), - такие переменные наз-ся свободными.
