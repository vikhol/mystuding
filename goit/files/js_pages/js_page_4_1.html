Глава 4

Массивы

Генератор фраз
Задание с пузырями

var scores = [60, 50, 60, 53, 47, 59, 50, 55, 53, 47, 59, 50, 55]; - массив (данных)
Можно работать с массивом как с единым целым, или же обращаться к отдельным значениям
var solution1 = scores[0]; - переменной solution1 присваивается значение стоящее на первом месте в массиве "scores". В нашем случае это число 60
Индексы массивов начинаются с нуля

В массивах можно перечислять названия каких-то "схожих" по смыслу элементов

Пример:
var icecream = ["Клубничное", "Ванильное", "Пломбир", "Эскимо"];
Чтобы выбрать нужный элемент из этого массива, нужно так же обратиться к его порядковому номеру.
Например, var bestIcecream = icecream[2]; - присвоит переменной bestIcecream значение "Пломбир".

в массивах могут храниться практически любые значения JavaScript, включая строки, числа, логические значения, другие массивы и даже объекты

Можно изменить любой параметр в массиве, обратившись к нему через индекс

Например:
var icecream[2] = "Шоколадное"; - изменит значение третьего по порядку эл-та в массиве (Пломбир) на "Шоколадное".
Присваивание значения элементу, не существующему в массиве, приведет к созданию нового элемента.
var numIcecream = icecream.length; - используется для выяснения кол-ва эл-тов в массиве.
var lastIcecream = icecream.length - 1; - используется для выяснения индекса последнего эл-та в массиве.

Создание пустого массива (и добавление элементов).

var genres = []; - так создается пустой массив, для последующего заполнения.

добавлять эл-ты в этот м-сив можно присваивая значения к какому-то конкретному индексу:


genres[0] = "Rockabilly";
genres[1] = "Ambient";

Но лучше воспользоваться методом push

var genres = [];
genres.push("Rockabilly"); - Создает новый элемент по следующему доступному индексу (который равен 0) и присваивает ему значение “Rockabilly".
genres.push("Ambient"); - Создает еще один элемент в следующей свободной позиции (1 в данном случае) и присваивает ему значение “Ambient".

Для перебора элементов массива обычно используется цикл for. Цикл while чаще всего используется, когда количество итераций неизвестно, и цикл должен выполняться, пока условие остается истинным.
