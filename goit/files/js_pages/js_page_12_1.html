Глава 12

Создание объектов.

Объектный литерал - полное описание объекта.

Конструкторы.

Основа - constructor.js
Пример с кофе - exerciseCoffee.js
Пример с тачками - exerciseCars.js
Альтернативный пример с тачками - exerciseCars2.js

  • Обычно, имя конструктора начинается с заглавной буквы, но это просто соглашение, а не правило.
  • Имена свойств и параметров так же не обязательно должны совпадать, это просто еще одно соглашение для удобства.

  • В кон-ры можно добавлять методы, которые будут автоматически добавляться к новым объектам
  • Объекты, созданные конструкторами, можно потом изменять: добавлять новые св-ва и методы, удалять ненужные. Пример:
    var fido = new Dog("Fido", "Mixed", 38);
    fido.owner = "Bob";
    delete fido.weight;

  • Для создания объекта, используется оператор "new".
    var houseBlend = new Coffee("House Blend", 12);

  • В параметры конструктора можно передавать готовый объектный литерал. Это очень удобно если в конс-ре помимо свойств, имеются еще методы и одинаковые для всех объектов параметры св-в. (Пример в exerciseCars2.js)

    • instanceof

    instanceof - позволяет проверить, создан ли объект конкретным кос-ром, указаным в условии.

  • Этот оператор проверяет не только конкретный конструктор, но так же учитывает все конструкторы, от которых что-то наследуется для создания объекта.

    • Встроенные конструкторы.

    standartConst.js

    Date - конструктор даты. Располагая таким объектом, можно использовать его методы для выполнения операций с датой и временем.
    var now = new Date();

  • toString() - метод, возвращающий полную дату и время в виде строки: “Thu Jun 08 2017 11:56:23 GMT+0700”.
    var dateString = now.toString();
  • getFullYear() - метод, возвращающий текущий год.
    var theYear = now.getFullYear();
  • getDay() возвращает номер дня недели (1 - понедельник)
