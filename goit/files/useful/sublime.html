Горячие кнопки

  • Ctrl + Shift + G - оборачивает выделенный фрагмент в нужный тег.
  • Ctrl + Shift + H - форматирует код, для более удобного чтения. (Нужно установить HTML-CSS-JS Prettify.)
  • Ctrl + H - ищет все совпадения с выделенным фрагментом и позволяет заменить сразу все найденные совпадения на новое значение.
  • Ctrl + Alt + Shift + M - сожмет код CSS (JS больше не поддерживается) и сохранит в новом файле. (Требуется плаг Minifier)

    • Плагины.

    Установка:
    1. Ctrl + Shift + P - Открывает Package Control (если он установлен ).
    2. Вводим/выбираем пункт Install Package
    3. Вводим название нужного плагина.

  • All Autocomplete - автозаканчивание.
  • Emmet - плаг для краткого написания кода. Шпаргалка по Emmet.
  • Autoprefixer - атоматически проставляет префиксы. Видео по настройке плага.
  • HTML-CSS-JS Prettify - плаг для форматирования кода в более читаемый вид. Ctrl + Shift + H
  • AutoFileName - добавляет меню для выбора месторасположения подключаемого файла.
  • jQuery - плаг для более удобной работы с jQuery. Видео по плагу.
  • Minifier - сжимает CSS (JS больше не поддерживается) код. Ctrl + Alt + Shift + M - сожмет код и сохранит в новом файле.
  • CSS Unminifier - разворачивает минифицированный CSS в удобочитаемый. Легко заменяется плагином HTML-CSS-JS Prettify (Ctrl + Shift + H).
  • Live Reload - автообновление редактируемых страниц. Установка и использование.
