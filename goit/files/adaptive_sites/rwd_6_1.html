Создание теней для текста средствами CSS3

  • text-shadow - св-во, отвечающее за тень текста.

    • Пример текста с тенью.

  • В кратком описании св-в задаются несколько параметров:
    1. Смещение по вертикали
    2. Смещение по горизонтали
    3. Размытие
    4. Цвет тени

    • В итоге CSS-правило выглядит следующим образом: text-shadow: 4px 4px 2px #dad7d7;

  • Можно задать несколько теней, разделив значения запятой(ми). Например, text-shadow: 0px 1px #fff, 4px 4px 0px #dad7d7;

  • Если размытие не нужно, то его можно и вовсе не записывать.

  • Так же можно задать размеры используя rem, а для цвета воспользоваться rgba.

  • Для эстетической привлекательности лучше все-таки пользоваться размерами в пикселях и задавать значения в пределах 1-2 px.

    • Тени для блоков box-shadow

  • Тени для блоков работают по тому же принципу, и имеют те же параметры, что и тени текста (см. выше) + параметр протяженности/распространения (spread).
  • Для блоков можно задать внутреннюю тень. Для этого значения должны начинаться со слова inset

    • Тень снаружи.

    box-shadow: 0px 3px 5px -6px #444;

    Тень внутри.

    box-shadow: inset 0 0 10px #444;

  • Если задать параметру протяженности/распространения (spread) отрицательное значение, то тень как бы задвинется внутрь, под объект (пример выше - 1 блок).

    • Использование нескольких фоновых изображений

    Пример

    В одном блоке можно использовать сразу несколько фоновых изображений.

    Они записываются через запятую:

    .bg {

     background:

     url('../img/1.png'),

     url('../img/2.png'),

     url('../img/3.png');

    }

  • Первое в списке изображение будет самым верхним.

  • Чтобы установить индивидуальные размеры для каждого изображения, нужно воспользоваться св-вом background-size
  • Значения так же записываются через запятую. Порядок записи будет соответствовать порядку изоб-й в списке background.
  • Для каждого изображения указывается сперва ширина, а затем, через пробел, высота.

    • Индивидуальные размеры для нескольких фоновых изображений. Для каждого изображения указывается сперва ширина, а затем, через пробел, высота.

    .bg {

     background-size: 100% 50%, 300px 400px, auto;

    }

    При присваивании размеров можно воспользоваться, как привычными значениями в виде процентов и пикселей, так и следующими ключевыми словами:

  • auto - задает естественный размер изображения.
  • cover - расширяет изображение, сохраняя соотношение его сторон, чтобы занять всю площадь элемента.
  • contain - расширяет изображение таким образом, чтобы оно поместилось внутри элемента по своей самой длинной стороне с сохранением соотношения сторон.

    • CSS-фильтры

  • filter: blur(); - размытие. Задается в пикселях или др. величинах (но только не в процентах)
  • filter: brightness();; - используется для затемнения изображения. Задается значениями в диапазоне от 0 до 1, или в процентах. 1 и 100% обеспечивают дефолтную яркость. Если выкрутить более 100%, то изображение станет светлее.
  • filter: contrast(); - контраст. Применяются значения в диапазоне от 0 до 1 или от 0 до 100 %. 1 или 100% обеспечивают нормальную контрастность, а более высокие значения вызывают повышение цветовой контрастности.
  • filter: grayscale(); - обесцвечивание. Используются значения от 0 до 1 или от 0 до 100 %, приводящие к различным степеням обесцвечивания элемента. Значение 0 оставляет изображение полноцветным, а 1 его полностью обесцвечивает, превращая цвета в оттенки серого.
  • filter: hue-rotate(); - изменение цвета по цветовому кругу. Используются значения от 0 до 360°.
  • filter: invert(75%); - инвертирование цвета. Используются значения от 0 до 100%.
  • filter: opacity(50%); - прозрачность. От 0 до 100%.
  • filter: saturate(); - сатурация эл-та. Исчисляется в процентах, может быть более 100%.
  • filter: sepia(.75); - сепия. Используются значения от 0 до 1, или от 0 до 100%.
  • filter: drop-shadow(-14px -18px 6px #E4E57C); - добавляет тень к эл-ту (а не ко всему блоку). Охуенно работает с png картинками.
