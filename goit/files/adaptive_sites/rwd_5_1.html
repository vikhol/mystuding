    • Запросы возможностей (feature queries).

    @supports позволяет задать условия выполнения объявления, основанные на возможностях конкретного браузера.

    Пример 1

    Если браузер поддерживает технологию flex, то выполняется код объявления.

    @supports (display: flex) {

    .Item {

    display: inline-flex;

    }

    }
    Пример 2

    Если браузер не поддерживает технологию flex, то выполняется код объявления.

    @supports not (display: flex) {

    .Item {

    display: inline-block;

    }

    }

    В примерах выше один блок кода определяется на случай поддержки возможности, а другой — на случай отсутствия такой поддержки.

    Комбинирование условий

    Можно задать сразу несколько условий!

    Пример 1

    Строгое соответствие.

    @supports ((display: flex) and (pointer: coarse)) {

    .Item {

    display: inline-block;

    }

    }
    Пример 2

    Строгое соответствие + or

    @supports ((display: flex) and (pointer: coarse)) or (transform:translate3d(0, 0, 0)) {

    .Item {

    display: inline-block;

    }

    }

  • @supports вообще не поддерживается в IE, поэтому лучше использовать Modernizr.

    Новые селекторы в CSS3 и порядок их использования

    Селекторы атрибутов в CSS3 (c.126)

  • img[alt] { width: 20rem; } - применит стиль к каждому img, у которого есть атрибут alt.
  • img[alt="sausages"] { width: 20rem; } применит стили только к тем эл-там img, которые имеют атрибут alt со значением sausages.
  • [alt] { width: 20rem; } - применит стили к любым эл-там, у которых имеется атрибут alt.
  • [alt*="cream"] { // Cтили }. - В этом примере, * означает, что атрибут "содержит" в себе это слово. Например <img src="п/к/ф" alt="this cream pie">
  • img[alt^="film"] { // Стили } - селектор по подстроке атрибута, которая находится в начале.
  • [alt$="jam"] { // Стили } - селектор по подстроке находящейся в конце.

    • Структурные псевдоклассы CSS3

  • :first-child и :last-child (просто чтобы не забыть)

    • nth-child

    :nth-child используется для добавления стиля к элементам на основе нумерации в дереве элементов.
  • может иметь следующие значения:
    1. odd - все нечетные номера элементов.
    2. even - все четные номера элементов.
    3. число - порядковый номер дочернего элемента относительно своего родителя. Нумерация начинается с 1, это будет первый элемент в списке.
    4. выражение - задается в виде an+b, где a и b целые числа, а n — счетчик, который автоматически принимает значение 0, 1, 2... Например, 3n+2 выберет 2, 5, 8, 11, 14 и т.д.

  • :nth-last-child(-n+3) - сперва выберет третий эл-т с конца, а затем все эл-ты после него.

    • Селектор отрицания (:not)

    div:not(.not-me) { // Стили } - применит стили ко всем эл-там div, которые не содержат класс .not-me

    Еще селекторы

  • :first-line - используется для изменения цвета первой строки, видимо на экране. При изменении ширины экрана, первая строка так же будет изменяться.
  • :first-letter - первая буква.

    • CSS-функция calc

  • calc() - функция, используемая для арифметических операций в коде CSS. Например, .thing { width: calc(50% - 10px); } - задаст для thing ширину 50% и вычтет из нее 10px. Не работает в Android 4.3 и ниже.
  • Поддерживаются сложение, вычитание, деление и умножение.

    • Адаптивные меры длины (vmax, vmin, vh, vw)

    В CSS имеются единицы измерения, выражаемые относительно размеров окна просмотра. Каждая единица измерения представляет собой процентное отношение к длине окна просмотра.

  • vh - для высоты окна просмотра.
  • vw - для ширины окна просмотра.
  • vmin - для минимального размера окна просмотра, равна самому малому значению — либо vw, либо vh;
  • vmax - для максимального размера окна просмотра, равна самому большому значению — либо vw, либо vh.

    • CSS-правило @font-face

    1. Сперва объявляется @font-face и открываются фигурные скобки.
    2. В них записывается название шрифта, например, font-family: Capture_it;
    3. И задается путь к файлу шрифтов. Например, src: url(../fonts/Capture_it.ttf);
