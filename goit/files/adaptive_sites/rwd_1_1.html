Основы адаптивного веб-дизайна

Начинать разработку сайта стоит всегда с представления для самого маленького экрана.
1rem = 16px

Мета-тег viewport - область просмотра браузера

<meta name="viewport" content="width=device-width">
  • Добавляется в контейнер head и предписывает браузеру отобразить содержимое страницы во всю ширину экрана устройства.

    • Масштабирование картинок

  • Для правильного масштабирования картинок на мобильных устройствах, используется CSS св-во max-width.
    • img { max-width: 100%; }
  • Значение max-width задается в процентах.
  • Привычный width: 100%; НЕ ПОДХОДИТ, так как он служит только для заполнения своего контейнера по всей его ширине.

    • Ввод медиазапросов.

    Медиазапросы позволяют применять конкретные CSS-правила на основе целого ряда условий (например, ширины и высоты экрана).
    НЕ УСТАНАВЛИВАТЬ ЗНАЧЕНИЯ КОНТРОЛЬНЫХ ТОЧЕК РАВНЫМИ ШИРИНЕ ПОПУЛЯРНЫХ УСТРОЙСТВ

    Потому что нам нужен именно адаптивный дизайн, а не дизайн для конкретных устройств.
